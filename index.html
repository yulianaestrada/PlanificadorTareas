Planificador de Tareas

Este proyecto consiste en un planificador de tareas diseñado para ayudar a organizar eventos o sesiones de estudio. Utiliza arrays para almacenar los eventos programados, como tareas, clases o recordatorios personales. Incluye funciones para programar dicha tarea y la duración de la misma, que ayuden a mantener el control del tiempo y las responsabilidades. Además, se definen variables específicas para gestionar horarios, materias, duración de las sesiones y prioridad de las tareas. La meta es ofrecer una herramienta simple y funcional para mejorar la productividad y la organización personal o académica.
