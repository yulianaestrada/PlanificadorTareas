Planificador de Tareas

Este proyecto es un planificador de tareas diseñado para organizar eventos o sesiones de estudio. Consume información de una API pública para mostrar las tareas registradas disponibles. Incluye funciones para programar cada tarea y definir su duración, facilitando el control del tiempo y las responsabilidades. Además, se utilizan variables específicas para gestionar horarios, actividades y la duración de las sesiones. El objetivo es ofrecer una herramienta simple y funcional que mejore la productividad y la organización personal o académica.
